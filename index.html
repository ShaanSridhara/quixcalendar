Calendar
Multi-calendar view
All your calendars are shown simultaneously, Apple Calendar-style. Toggle a calendar on/off by clicking its colored square in the sidebar. Click the calendar name to make it the active one (used when creating new events).
Month & Week views M W
Switch between Month and Week views with the buttons in the top bar or keyboard shortcuts. Week view shows timed events as blocks on a 24-hour grid.
Create an event
Click any calendar cell to open the new event form pre-filled with that date. In week view, click a time slot to pre-fill the time too. Use the + Meeting and + Task buttons in the top bar for quick creation.
Recurring events
Set any event to repeat daily, weekly, bi-weekly, monthly, or yearly. Add an optional end date to stop recurrence automatically.
Event types & colors
Meetings use the calendar color. Tasks are color-coded by priority — green = low, amber = important, red = critical. Completed events turn gray.
Mini calendar
The mini calendar in the sidebar shows dots for days with events. Click any date to jump to that month/week.
Search ⌘K
Full-text search
Search by event title, description, or participant name. Results are grouped into Meetings and Tasks. Click a result to jump directly to that date on the calendar and open the event details.
Tips
Search works across all your own and shared calendars. Try searching a teammate's name to find every event they're part of.
Dashboard
Team overview
The dashboard shows all events for a calendar week, grouped by member. Navigate weeks with the arrow buttons or jump back to the current week.
Attendance tracking
Admins and leaders can mark attendance directly from the dashboard or from an event's detail view. Attendance data is stored per-event.
Task completion
Task managers can mark tasks complete from the event detail view. Completed tasks appear grayed-out on the calendar and dashboard.
Admin & Sharing
Roles
Captain — full control, can rename/delete the calendar and manage all events.
Leader — can create and edit events, manage attendance.
Member — can view events they're invited to.
Sharing a calendar
Click the Share button in the calendar top bar. Share the invite code with teammates. They enter it in Admin → Shared Calendars to join. Joining auto-adds the shared calendar to their sidebar.
Rename a calendar
Double-click the calendar name in the top bar (Captains only) to rename it inline.
Post-meeting notes
After a meeting occurs, admins can add post-meeting notes to the event (visible in the detail view for all members).
Tips & Tricks
Double-click to edit
Double-click any event pill on the calendar to jump straight into the edit form, skipping the detail view.
Scroll to navigate
On the calendar page, scroll up or down with your mouse wheel to move between months. No need to reach for the arrow buttons.
Filter by type
Use the Meetings / Tasks / All filter buttons in the calendar subbar to quickly see only what you need.
Toggle calendars
Click the colored square next to a calendar name to hide or show it. Hidden calendars don't clutter the view but are still in the sidebar, ready to re-enable.
Dark mode
Toggle between light and dark mode with the sun/moon icon in the top-right corner. Your preference is saved locally.
